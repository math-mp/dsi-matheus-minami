#  Desenvolvimento de software I
aulas de desenvolvimento de software com professor João Siles utilizando a linguagem de programação java.
# Linguagem java
java é uma linguagem orientada a objeto com funcionamento baseado em relações parentais de classes onde, uma classe engloba um conteudo que pode ser utilizado e referenciado mais para frente em outros codigos.
# Configuração do java no pc
Para configurar o java em uma nova maquina é preciso realizar a instalação de um kit de ferramentas para que o java possa compilar codigo em arquivos .jar chamado JDK (java development kit).

a recomendação é realizar a instalação via navegador a partir de distribuidoras de jdk como a open jdk, amazon corretto , open jdk da microsoft, ou adoptium.

>é importante ressaltar a crucialidade de uma etapa na instalação do java que é a seleção de todas as opçoes do instalador do JDK com hotspot.
#  Oque é JVM (java virtual machine)?
a JVM (java virtual machine) permite que o computador rode e interprete codigos compilados .jar em uma maquina virtual.
